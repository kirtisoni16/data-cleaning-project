# Data Cleaning and Visualization Project

## Overview
This project demonstrates basic data cleaning and data visualization using Python.

The dataset contains student marks and attendance records with some missing values. The project cleans the data and visualizes student performance using graphs.

---

## Technologies Used
- Python
- Pandas
- Matplotlib

---

## Features
- Handling missing values
- Data preprocessing
- Data visualization using bar charts
- Simple dataset analysis

---

## Dataset Example
| Student | Marks | Attendance |
|---------|------|------------|
| A | 85 | 90 |
| B | 90 | 95 |
| C | NaN | 85 |
| D | 78 | NaN |
| E | 92 | 88 |

---

## Output
The project generates a visualization graph of student marks after cleaning the dataset.
